# プロダクトのタイトル
DoKoDeMo DJ
〜福岡のパーティー絶対盛り上げる.com〜
## プロダクトの紹介

- 呼びたいと思ったらDJを呼ぶ事が出来るサービスのお問い合わせ画面と打合せ画面、進捗状況確認画面を作成しました。
- 打合せ内容がDBに送信され、イベントまでの準備の状況がいつでも確認が出来ます。
- 今後実装予定の機能
  ①Raspberry Piを使った誰でもガヤが表示できるLED掲示板の作成
  ②パーティーのイメージの共有ができる画面
  ③スマホ対応
