# プロダクトのタイトル
DoKoDeMo DJ
〜福岡のパーティー絶対盛り上げる.com〜
## プロダクトの紹介

- 呼びたいと思ったらDJを呼ぶ事が出来ます。
- 打合せ内容がDBに送信され、イベントまでの準備の状況がいつでも確認が出来ます。
- 今後実装予定の機能
  ・Raspberry Piを使った誰でもガヤが表示できるLED掲示板の作成
  ・パーティーのイメージの共有ができる画面
  ・スマホ対応

## 操作方法

- (ユーザー側)
   TOP → お問い合わせ送信 → 打合せの内容と準備の状況が確認できる

- (運営側)
  ログイン → お問い合わせ一覧に表示 → (打合せ時)打合せ内容を入力 → 準備の状況の更新
